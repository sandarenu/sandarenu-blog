sandarenu-blog
==============

Hugo template for my personal blog (sandarenu.github.io)[sandarenu.github.io]

## Steps 

### Install themes

`git clone --depth 1 --recursive https://github.com/spf13/hugoThemes.git themes`
### Starting Server Locally

`hugo server --theme=casper`

### Building Site

`hugo --theme=casper`
